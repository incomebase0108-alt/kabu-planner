# 日本株 売買プランナー（試作）

`toushi/index.html` を**ブラウザで開くだけ**で動きます（スマホ可）。サーバー不要・**トークン消費ゼロ**。

## 使い方
1. 銘柄コード（例: `7203` トヨタ、`186A` アストロスケール）を入れて「実行」
2. スタイル（短期／中期／長期）を選ぶと利確・損切りの幅が変わる
3. 表示されるもの：
   - 現在値・前日比
   - 短期／長期トレンド判定（移動平均線ベース）
   - RSI（買われすぎ／売られすぎ）
   - **買い目安・利確ライン・損切りライン**（ATR＝値動きの大きさから自動算出）
   - リスク／リワード比

## 自動取得が弾かれたら
ブラウザのCORS制限や通信環境で株価取得に失敗することがあります。その場合は
「手動で現在値を入れる」を開き、現在値（＋任意で直近高値・安値）を入れれば
利確・損切りは計算できます。

## 注意
- これは過去の株価から機械的に計算した**目安を表示する道具**であり、値上がり・利益を**保証しません**。
- 銘柄が上がるかどうかは誰にも当てられません。損切りを機械的に守るための補助として使ってください。
- 投資は自己責任です。

## データ源
Yahoo Finance の公開chart API（`{コード}.T`）。直接取得が不可のときは
公開CORSプロキシ経由で自動リトライします。遅延・欠損の可能性あり。
